# Хуваагч
**1**-ээс **N** хүртлэх тооноос хамгийн их хуваагчтай тоог ол.

# Input format
N тоог илэрхийлэх ганц бүхэл тоо өгөгдөнө.

# Constraints
**N &le; 10000**

# Output
Хариу болох ганц бүхэл тоог хэвлэнэ.

# Sample testcases
##### Sample Input 0
```
13
```
##### Sample Output 0
```
12
```
